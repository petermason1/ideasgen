# ideasgen static launch bundle

This folder contains the standalone HTML pages we ship publicly:
- `index.html` – launch hub linking every artifact
- `dashboard.html` – quick directory of links
- `secure-features-landing.html` – Secure AI Control stack sales sheet
- `secure-launch-studio.html` – consultancy landing page
- `trade.html` – trade registration gate
- `dream.html` – Dream Creator upsell
- `godmode-runner.html` – internal console (kept here for completeness)
- `web-dev-quotes.html` – sample signal remix
- `control-console.html` – founder control console mock
- `styles.css` – shared styling

Everything else in the main project stays private.
